About

Welcome. I'm a developer at VMware (formerly Pivotal Labs) and currently a MS student at NYU studying Data Science.

This is a place to share data and computing related work. The goal is to provide ideas for readers in their own work as well as for me to learn from the engagement and ideas of others in this community.

This site is hosted on Amazon EKS using a docker image I created to run a Spring Boot application. I hope to eventually move all of my computing services off of Amazon because of their ethics as a company (look for more details in a future post :), but for now this is what we have.

Some of the code for posts can be found at my github: gholden3. I plan on linking the github repository with each blog post, where possible. Unfortunately much of the code I discuss in posts is hosted in private repositories but I will attempt to pull out shareable chunks of usable code and share those on public github as I work on the blog posts.

I hope to build out more functionality on this site in the future, but for now enjoy the blog and feel free to get in touch: gina.m.holden at gmail.com (unless you are a recruiter from facebook or amazon in which case please do not email me).
